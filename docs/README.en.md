# WebStyle Kit - English

Modern userstyle collection to customize and enhance the web experience.

## Structure
Each style is standalone and can be used independently via Stylus or similar extensions.

## Purpose
WebStyle Kit provides lightweight userstyles to enhance, modernize, and customize web interfaces without breaking native UI behavior. Includes modified Stylus themes and forks such as Liquid Glass.

## Compatibility
- Stylus (recommended)
- Chrome / Edge / Firefox
