# 🎨 WebStyle Kit

A lightweight collection of modern userstyles designed to enhance and customize the web experience without breaking native UI behavior.

---

## ✨ Overview

WebStyle Kit is a modular set of userstyles focused on aesthetics, usability, and consistency.  
Each style integrates seamlessly into existing websites, improving visual experience without altering core functionality.

---

## 📦 Structure

Each style is fully independent and can be installed or removed without dependencies.

```
styles/
  └── example-style.css
  └── another-style.scss
  └── ...
```

---

## 🚀 Purpose

- Modernize web interfaces  
- Improve visual consistency across platforms  
- Add high-quality UI enhancements  
- Support custom Stylus-based themes and forks  
- Keep styles lightweight and non-intrusive  

---

## 🔧 Compatibility

- Stylus (recommended)
- Chrome / Edge / Firefox
- Compatible with most modern browsers

---

## 🎨 Features

- Lightweight CSS architecture  
- No external dependencies  
- Modular style system  
- Safe UI overrides  
- Theme-friendly design (dark/light support)

---

## 🛡️ Philosophy

Enhance, don’t replace.
